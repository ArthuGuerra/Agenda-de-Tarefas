Sistema de Gerenciamento de Tarefas (To-Do List) - Versão Avançada
Descrição
O sistema de gerenciamento de tarefas permitirá ao usuário organizar tarefas diárias com recursos como adicionar, editar, excluir, e categorizar tarefas, além de integração com um banco de dados para persistência. Pode ser uma aplicação console ou uma aplicação web simples com interface gráfica.

Funcionalidades
Adicionar tarefa:

O usuário pode criar uma nova tarefa informando: descrição, data de vencimento e prioridade (alta, média, baixa).
Categorias: O usuário pode atribuir categorias (trabalho, pessoal, estudo) para melhor organização.
Editar tarefa:

O usuário pode alterar qualquer informação de uma tarefa (descrição, data de vencimento, prioridade, categoria).
Excluir tarefa:

O usuário pode remover tarefas concluídas ou não necessárias, ou até mesmo limpá-las automaticamente após certo período.
Listar tarefas:

Filtrar por status: Mostrar tarefas concluídas e pendentes separadamente.
Filtrar por categoria ou prioridade: O usuário pode visualizar as tarefas filtradas por categoria (trabalho, pessoal, estudo) ou por prioridade.
Marcar como concluída:

Permite marcar uma tarefa como concluída. Tarefas concluídas podem ser arquivadas.
Persistência de dados:

As tarefas são salvas em um banco de dados local (SQLite, por exemplo) ou arquivo XML/JSON. Isso permite que as tarefas sejam mantidas entre sessões de uso.
Relatórios de Progresso:

O sistema pode exibir um relatório com estatísticas, como o percentual de tarefas concluídas e o tempo médio para completar uma tarefa.
Notificações:

Notificação de prazo: Exibir uma notificação de alerta quando o prazo de uma tarefa estiver próximo ou for ultrapassado (pode ser via console, ou email, se for web).
Interface Gráfica Opcional (UI):

Caso você queira expandir, pode implementar uma interface gráfica (via WinForms, WPF, ou uma versão web em ASP.NET Core). Se preferir algo mais simples, pode começar pelo terminal.
Recursos que você aprenderá:
Manipulação de listas, coleções e estruturas de dados.
Persistência de dados em banco de dados ou arquivos XML/JSON.
Manipulação de datas e tempos (para prazos).
Operações CRUD (Create, Read, Update, Delete).
Como criar filtros e relatórios dinâmicos.
Como integrar C# com uma interface gráfica (WinForms, WPF, ASP.NET).
